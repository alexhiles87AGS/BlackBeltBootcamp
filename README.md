# BlackBeltBootcamp V2.1.2

Mobile UI polish build. This update keeps the V2.1.1 app structure and working features, but improves the iPhone experience and changes FMA class handling so classes are logged as class sessions, not individual exercise workouts.

## Included

- iPhone-optimised visual polish and responsive layout
- compact mobile navigation
- cleaner mobile cards, panels, buttons and forms
- FMA class sessions added as attendance events only
- no individual exercise log required for FMA classes
- existing exercise library, video modal, instructions tab, workout builder, dashboard, calendar, profile, stats, badges and import features retained

## Deploy

Copy these files over the existing BlackBeltBootcamp project, run `npm install`, then `npm run build`, commit and push to GitHub.
